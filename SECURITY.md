# Security Policy

## Reporting a Vulnerability

**Please report vulnerabilities via GitHub's private vulnerability reporting feature** by going
[**here**](https://github.com/TeamSHIRO/KURO/security/advisories).

> [**Learn more about GitHub's private reporting process**](https://docs.github.com/en/code-security/security-advisories/guidance-on-reporting-and-writing-information-about-vulnerabilities/privately-reporting-a-security-vulnerability).


**What to Expect After Reporting:**

* We will acknowledge receipt of your report typically within **1–7 days**
* We will investigate the vulnerability promptly.
* We ask that you allow us a reasonable time to address the vulnerability
  before public disclosure.


> [!IMPORTANT]
> Please **do not** open a public GitHub issue for security vulnerabilities or publicly disclose them.

Your cooperation in responsible disclosure is highly valued and helps keep KURO secure for everyone.
